Description of Design and Meeting the Requirements

For my coursework project, I have extracted a smaller sample of data from a large, raw data file on this topic. I used the systematic sampling extraction method to do this because I could not mix up the different types of data being collected as they all came from different seas. I manually extracted all of my data from the section containing data from the Arabian Sea only. I then used Python to clean the raw dataset, my program removed any uneccessary symbols in cells and entirely removed any empty columns. My program then creates a new, clean csv file which I can then use for the rest of my project. I will then use the Python module pandas to assist me in carrying out analytics on my data; find the mean, median, mode and range of the dataset. I will use Python data structures such as dictionaries to manage the data I will be analysing. Furthermore, I will use Plotly graphing libraries to create visual representations of my data in the form of graphs. I will have a scatter plot to show the data distribution of the salinity and temperature in both January and June. I will then create two separate bar charts dealing with the salinity and tempereature of both months separately. To tie all aspects together, I will create a web-based interface that provides information on my coursework project to the public and presents all of my data visualisations and findings in an interactive mode.
